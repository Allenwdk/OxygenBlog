# 博客图片功能演示

这是一个演示新的博客文件夹结构和图片解析功能的示例文章。

## 新的文件夹结构

现在每个博客文章都可以使用文件夹形式存储：

```
blog-name/
├── index.md          # 博客内容
└── assets/           # 图片资源
    ├── image1.jpg
    └── image2.png
```

## 图片功能

在 Markdown 中，你可以使用相对路径引用图片：

```markdown
![示例图片](./example.svg)
![另一张图片](assets/another.png)
```

系统会自动将这些路径转换为正确的 API 路径。
